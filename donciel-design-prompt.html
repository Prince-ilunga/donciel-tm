1. Prompt Maitre — A copier-coller

Construis une application web professionnelle avec le design system suivant :

## PHILOSOPHIE DESIGN
- Design monochromatique neutre (noir/gris/blanc) comme base
- 4 couleurs semantiques domain-specific pour le trading : profit (vert), loss (rouge), gold (ambre), emerald (teal)
- Aucune couleur primaire huee (pas de bleu, violet, indigo par defaut)
- Aspect premium, epure, financier — inspiré des dashboards de trading professionnels
- Mode sombre par defaut, mode clair disponible
- Typographie : Geist Sans pour le texte, Geist Mono pour toutes les donnees numeriques
- Superscript TM apres le nom de marque
